# Turbine3D

3D turbine model with controllable blade speed

Parts
* Raspberry pi zero (minimum spec) or Arduino D1 Mini (Nodemcu)
* 6v dc model  motor
* L298N stepper controller
* Oled i2c lcd display (optional)
